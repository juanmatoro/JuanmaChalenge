# Chalenge de puntuación de Soporte

Desafío de componente de calificación interactivo realizado con HTML, CSS y JavaScript vanilla.

## Ejecutar

1. Abre `index.html` en el navegador.
2. O usa un servidor local (ejemplo con VS Code Live Server).

## Funcionalidades

- Vista inicial de calificación (1 a 5).
- Estados `hover` y `focus`.
- Selección visual del score.
- Envío y cambio a vista de agradecimiento con score dinámico.
